# Server Stat
Project consists typescript dependencies.

##Steps:

1. Install Project

```
npm install
npm run build
```

2. Start server

```
 npm run start:server
```

3. Start client

```
 npm run start:client
```

##DEVELOPMENT ENVIRONMENT

Start server

```
 npm run start:server:dev
```

Start client

```
 npm run start:client:dev
```
